# TestingGroundForAnalytics

TestingGroundForAnalytics

## Setup

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
2. Edit `.env` and fill in your `COC_API_KEY`, `CLAN_TAG` and `PORT` values.

Then install dependencies and start the server:
```bash
npm install
npm start
```
